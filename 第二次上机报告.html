问题描述

给定函数

分别利用复合矩形法则，复合梯形法则，复合Simpson法则求解的值，并将误差绘成关于的函数，比较三种数值积分的精度，判断误差是否有极限并说明。

实验环境

开发系统：OS X Yosemite 10.10.2

开发软件：Matlab R2014b

算法综述

实验要求构成关于h的函数，但是h表示的是将 区间等分后每个小区间的距离，那么显然h的变化不是线性的，如

因此，将误差关于h绘出的图像看上去基本为线性分段，所以将误差刻画成关于h的倒数的函数，既第n项为n等分后的误差。

矩形法则的算法则直接套用相应公式即可。

实验结果

图像展示

红色表示复合梯形法则的误差函数，

误差关于h的函数

由于h的非线性变化，
三种函数都呈现分段线性的样子。

误差关于h的倒数的函数

由于Simpson的区间从二等份开始，因此h最大为0.5。

结果分析

观察图像，三个函数随着h减小，既h的倒数增大，无限趋近于0。其中Simpson下降趋紧的最为平滑，梯形法则次之。

就精度而言，梯形法则的精度最高，矩形法则次之。

根据三种数值积分的误差函数，只要函数f(x)的一阶导数（矩形法则），二阶导数（梯形法则）和四阶导数（Simpson法则）在[0,1]区间内有界，则误差将随着h趋于0而趋于0。而该函数的无穷阶导在[0,1]区间内都是有界的（任意阶导数都由），因此误差最终趋于0。

代码展示

复合矩形法则

function [ RD ] = RectRule( f , h )
%UNTITLED5 Summary of this function goes here
%   Detailed explanation goes here
len = length(f);
RD = 0;
for i = 1 : (len-1)
    RD = RD + f(i)*h;
end

end

复合梯形法则

function [ TD ] = TrapezoidalRule( f , h )
%UNTITLED3 Summary of this function goes here
%   Detailed explanation goes here
len = length(f);
TD = 0;
for i = 1 : (len-1)
    TD = TD + (f(i)+f(i+1))*h/2;
end
end

复合Simpson法则

function [ SD ] = SimpsonRule( f , h )
%UNTITLED4 Summary of this function goes here
%   Detailed explanation goes here
len = length(f);
SD = 0;
for i = 1 : (len-2)
    SD = SD + (f(i) + 4 * f(i+1) + f(i+2) )*h/3;
    i = i + 1;
end
end
